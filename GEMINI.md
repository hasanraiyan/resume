# Project Instructions (GEMINI.md)

## Visuals & Diagrams

- **Mermaid Support**: This project supports Mermaid diagrams. You can (and should) design diagrams (flowcharts, sequence diagrams, class diagrams, etc.) using Mermaid syntax.
- **Rendering**: In the Coursify Search Engine and other components, Mermaid diagrams wrapped in \`\`\`mermaid\`\`\` code blocks are automatically rendered as interactive SVG visuals.
- **Best Practice**: Prefer Mermaid diagrams over ASCII art or text-based descriptions for technical concepts.
